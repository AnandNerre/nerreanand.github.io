# Nerre Anand Kumar Portfolio

Senior Data Engineer portfolio focused on Azure, Databricks, PySpark, SQL, and AI document processing.

## Open Locally

Open `index.html` in a browser. No build step is required.

## Files

- `index.html` - portfolio content and SEO metadata
- `styles.css` - responsive visual design
- `script.js` - mobile menu, project search, reveal animations, email draft form
- `assets/data-platform-hero.png` - hero visual

## GitHub Pages

For a personal GitHub Pages site, the repository should usually be named:

```text
<your-github-username>.github.io
```

After pushing the files, enable Pages from the repository settings:

```text
Settings -> Pages -> Deploy from a branch -> main -> root
```

This project is static and can also be deployed on Netlify or Vercel.
